# Day 10 - Advanced SQL

## Overview

This repository contains my Day 10 SQL assignment on Advanced SQL concepts. The queries are written using the Online Retail dataset and demonstrate different SQL techniques such as CTEs, Window Functions, CASE statements, Subqueries, and Date Functions.

## Dataset

- Online Retail Dataset

## Topics Covered

- Common Table Expressions (CTE)
- Window Functions (RANK)
- Running Total
- CASE Statement
- Subquery in WHERE clause
- Subquery in SELECT clause
- Date Functions
- Combined Query (CTE + Window Function)

## Files

- `day10_sql_advanced.sql` – SQL file containing all 8 advanced SQL queries.
- `online_retail.csv` – Dataset used for the assignment.
- `README.md`

## Queries Included

1. CTE to calculate summary and query the result.
2. Rank records within a group using a window function.
3. Calculate a running total.
4. Categorize records using a CASE statement.
5. Use a subquery in the WHERE clause.
6. Use a subquery in the SELECT clause.
7. Extract month and year using date functions and group the data.
8. Combine CTE and Window Function in a single query.
